

Accounting of Shop - Java Enterprise Online Project 
====================================================

Разработка полнофункционального Spring/JPA Enterprise приложения c авторизацией и правами доступа на основе ролей с использованием наиболее популярных инструментов и технологий Java: Maven, Spring MVC, Security, JPA(Hibernate), REST(Jackson), Bootstrap (css,js), datatables, jQuery + plugins, Java 8 Stream  и хранением в базах данных MYSQL, Postgresql и HSQLDB.
